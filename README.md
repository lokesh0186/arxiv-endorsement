# arxiv endorsement protocol.

thanks to Gen AI, the arXiv community is growing and I receive lots of endorsement requests.

if you want me to endorse you on arxiv:

1. read https://www.monperrus.net/martin/arxiv-endorsement
2. Make a pull-request on https://github.com/monperrus/arxiv-endorsement/, see example at https://github.com/monperrus/arxiv-endorsement/blob/main/requests/your-name.txt
